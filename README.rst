Open edX Webhooks
=================

Webhooks for `Open edX`_ integrating `JIRA`_ and `Github`_. Designed to
be deployed onto `Heroku`_.

|Build Status| |Coverage Status| |Documentation badge|

How to Run Tests
----------------

.. code:: sh

    pip install -r dev-requirements.txt
    make test
